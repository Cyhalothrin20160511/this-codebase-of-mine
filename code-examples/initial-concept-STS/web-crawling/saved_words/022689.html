Ετυμολογία [επεξεργασία]

δέντρο < (κληρονομημένο) μεσαιωνική ελληνική δέντρο(ν) < (κληρονομημένο) αρχαία ελληνική δένδρον που προφερόταν με [nd][1] < πρωτοϊνδοευρωπαϊκή *der-drew- < *dóru (δέντρο)

Προφορά[επεξεργασία]

ΔΦΑ : /ˈðen.dɾo/
τυπογραφικός συλλαβισμός : δέ‐ντρο

Ουσιαστικό[επεξεργασία]

δέντρο ουδέτερο

  1. (βοτανική) φυτό του οποίου ο κορμός έχει κλαδιά και φύλλα από ένα ορισμένο ύψος και πάνω
    ↪ Η φλαμουριά είναι ένα ωραίο δέντρο.
  2. (μεταφορικά) γραφική απεικόνιση που παρουσιάζει ανάλογα χαρακτηριστικά
    ↪ γενεαλογικό δέντρο
  3. (πληροφορική) αφηρημένος τύπος δεδομένων, με ιεραρχική δομή, όπου κάθε κόμβος έχει μηδέν ή περισσότερους κόμβους ως παιδιά
    υπώνυμα: δυαδικό δέντρο
