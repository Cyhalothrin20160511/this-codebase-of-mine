Ετυμολογία [επεξεργασία]

δημοψήφισμα < δημο- + ψήφισμα (μεταφραστικό δάνειο από τη γαλλική plébiscite) [1]

Προφορά[επεξεργασία]

ΔΦΑ : /ði.moˈpsi.fi.zma/
τυπογραφικός συλλαβισμός : δη‐μο‐ψή‐φι‐σμα

Ουσιαστικό[επεξεργασία]

δημοψήφισμα ουδέτερο

  • (πολιτική) ο θεσμός κατά τον οποίο το εκλογικό σώμα καλείται να εκφραστεί με ένα ναι ή ένα όχι σε μία ερώτηση για σημαντικό ζήτημα που θέτει η εκτελεστική εξουσία

Συνώνυμα[επεξεργασία]

  • ρεφερέντουμ

Συγγενικά[επεξεργασία]

  • δημοψηφισματικός
  • → δείτε τις λέξεις δήμος, ψηφίζω και ψήφος
