Ετυμολογία [επεξεργασία]

μισώ < μισῶ < αρχαία ελληνική μισέω, -ῶ < μῖσος

Ρήμα[επεξεργασία]

μισώ

  • έχω εχθρική διάθεση εναντίον ενός ανθρώπου ή κατάστασης, απεχθάνομαι κάποιον ή κάτι

Συνώνυμα[επεξεργασία]

  • τρέφω μίσος (για κάποιον) : εχθρεύομαι / μισώ κάποιον
  • σιχαίνομαι
  • απεχθάνομαι
