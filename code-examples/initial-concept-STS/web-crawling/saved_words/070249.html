Προφορά[επεξεργασία]

ΔΦΑ : /poˈli.ɣo.no/
τυπογραφικός συλλαβισμός : πο‐λύ‐γω‐νο

Ουσιαστικό[επεξεργασία]

πολύγωνο ουδέτερο

  • (γεωμετρία) γεωμετρικό σχήμα που αποτελείται από μια κλειστή τεθλασμένη γραμμή
