Ουσιαστικό[επεξεργασία]

πίθηκος αρσενικό

  1. (θηλαστικό ζώο) πρωτεύων θηλαστικό που ανήκει στην υπεροικογένεια των Aνθρωποειδών (π.χ. γίββωνας, χιμπατζής, ουραγκοτάγκος, άνθρωπος).
  2. (μεταφορικά) άσχημος άντρας, συνήθως πολυ τριχωτός
