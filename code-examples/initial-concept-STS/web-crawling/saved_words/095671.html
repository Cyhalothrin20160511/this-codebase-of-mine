Ετυμολογία [επεξεργασία]

χρηματοδοτώ < (διαχρονικό δάνειο) μεσαιωνική ελληνική χρηματοδοτῶ < αρχαία ελληνική χρῆμα + -δοτῶ (< δίδωμι). Μορφολογικά αναλύεται σε χρηματο- + -δοτώ

Προφορά[επεξεργασία]

ΔΦΑ : /xɾi.ma.to.ðoˈto/
τυπογραφικός συλλαβισμός : χρη‐μα‐το‐δο‐τώ

Ρήμα[επεξεργασία]

χρηματοδοτώ, αόρ.: χρηματοδότησα, παθ.φωνή: χρηματοδοτούμαι, μτχ.π.ε.: χρηματοδοτούμενος, π.αόρ.: χρηματοδοτήθηκα, μτχ.π.π.: χρηματοδοτημένος

  • παρέχω σε οργάνωση ή άτομο χρήματα για ορισμένο έργο ή σκοπό

Πηγές[επεξεργασία]

  • χρηματοδοτώ - Χαραλαμπάκης, Χριστόφορος (επιμέλεια) (2014). Χρηστικό λεξικό της νεοελληνικής γλώσσας. Αθήνα: Ακαδημία Αθηνών.  (ψηφιοποιημένη έκδοση από το 2023, συντομογραφίες-σύμβολα)
