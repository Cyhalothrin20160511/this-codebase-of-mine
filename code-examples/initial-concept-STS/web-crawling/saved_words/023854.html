Ετυμολογία [επεξεργασία]

διαολίζω < διάολ(ος) + -ίζω

Προφορά[επεξεργασία]

ΔΦΑ : /ðʝa.oˈli.zo/
τυπογραφικός συλλαβισμός : δια‐ο‐λί‐ζω

Ρήμα[επεξεργασία]

διαολίζω, αόρ.: διαόλια, παθ.φωνή: διαολίζομαι, π.αόρ.: διαολίστηκα, μτχ.π.π.: διαολισμένος

  • (οικείο) πικάρω, εξοργίζω, δαιμονίζω
    ※  Κοργιοί, κουνούπια, σκνῖπες, ψύλλοι καὶ μερμήγκοι, / Νά σε τσιμποῦν φρικτὰ καὶ νά σε διαολίζουν
    Κωνσταντίνος Σκόκος, «Το καλοκαίρι εις την Αθήνα», στο Γελοιογραφικόν Ημερολόγιον του Έτους 1886
