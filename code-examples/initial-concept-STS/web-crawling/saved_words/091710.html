Ετυμολογία [επεξεργασία]

φθινόπωρο < αρχαία ελληνική φθινόπωρον φθίνω (=λιγοστεύω) + ὀπώρα (=φρούτο)

Ουσιαστικό[επεξεργασία]

φθινόπωρο ουδέτερο

  • μία από τις εποχές του έτους
    άνοιξη, καλοκαίρι, φθινόπωρο, χειμώνας
