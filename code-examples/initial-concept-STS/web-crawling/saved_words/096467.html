ψέγω < (διαχρονικό δάνειο) αρχαία ελληνική ψέγω

Προφορά[επεξεργασία]

ΔΦΑ : /ˈpse.ɣo/
τυπογραφικός συλλαβισμός : ψέ‐γω

Ρήμα[επεξεργασία]

ψέγω, αόρ.: έψεξα, παθ.φωνή: ψέγομαι, π.αόρ.: ψέχτηκα

  • επικρίνω, ασκώ αρνητική κριτική, κατακρίνω, μέμφομαι
