Ετυμολογία [επεξεργασία]

κλείω < (διαχρονικό δάνειο) αρχαία ελληνική κλείω

Προφορά[επεξεργασία]

ΔΦΑ : /ˈkli.o/
τυπογραφικός συλλαβισμός : κλεί‐ω
τονικό παρώνυμο: Κλειώ

Ρήμα[επεξεργασία]

κλείω

  • (απαρχαιωμένο) κλείνω
    ※  Τον πλένει από τες σκόνες κι απ' τα αίματα·
    κλείει τες φοβερές πληγές, μη αφίνοντας
    κανένα ίχνος να φανεί· […]
    Κωνσταντίνος Καβάφης, Η κηδεία του Σαρπηδόνος
    απαντά σε σύνθετα ρήματα

Ετυμολογία 3[επεξεργασία]

κλείω επικός τύπος < κλέω < καλέω. Δείτε και κλῄζω

Ρήμα[επεξεργασία]

κλείω

  • επικός τύπος του κλέω (με σημασία: "καλώ"), άλλης μορφής του καλέω

Σύνθετα[επεξεργασία]

  • μετακλείω ("καλώ με νέο όνομα")
